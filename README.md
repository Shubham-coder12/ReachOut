ReachOut 🚀

​ReachOut is an elegant, full-stack inquiry management platform featuring a sleek dark-themed frontend contact form and a secure, password-protected backend administration dashboard. Built with a robust Python and Flask backend, it streamlines user communications while giving administrators total oversight of incoming requests.
​✨ Features
​Sleek Frontend Contact Form: A clean UI featuring real-time interactive inputs for Name, Email, Topic Selection (e.g., General Inquiry, Freelance, Collabs), and a character-limited message box.
​🔒 Secure Admin Authentication: A dedicated password-protected login portal ensures sensitive inquiry details are restricted from unauthorized access.
​📊 Analytics Dashboard: The admin panel tracks message metrics at a glance, using dedicated cards to count Total Messages, Freelance inquiries, and Collab requests.
​⚙️ Complete Data Control: Administrators can easily review message details, check precision UTC timestamps, and delete entries to keep the storage clean.
